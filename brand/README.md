# Caucus brand

The mark: **three voices (the panel) converging into one (the synthesis)** — the product in one
glyph. Use it everywhere: the console header, the favicon, the marketing demo, READMEs, and (when a
LiteLLM Proxy is attached for the team case) wherever a brand mark is allowed.

## Files
- `caucus-mark.svg` — the mark, `fill: currentColor`. Set `color` to the brand blue (or cream on a
  dark surface). The reusable one.
- `caucus-icon.svg` — the mark in fixed brand blue on transparent. For favicons / app icons.

## Palette
| token | hex | use |
|---|---|---|
| blue | `#2f6fd6` | the mark + accents on light (cream) surfaces |
| blue (dark) | `#5ea2ff` | the mark + accents on dark (navy) surfaces |
| cream | `#f4edda` | light background |
| navy | `#0e1622` | dark background |
| ink | `#1f2c3d` | wordmark text on cream |

## Wordmark
Lowercase `caucus`, the sans at weight 500, letter-spacing `-0.5`. Sentence case — never ALL CAPS.

The mark is `currentColor`, so it inherits the surrounding text color; that's how it works in both
light and dark mode with no second asset.
