Get set up three steps to your first synthesized turn

    Providers

    Add an API key for each provider your combos use. BYOK — keys are stored locally in ~/.config/caucus/.env, sent only to the provider you chose, and only a fingerprint is ever shown back.

    Combos

    A combo is a panel of models + a judge + a strategy. Plan turns convene the panel and the judge synthesizes one answer; action turns pass through to the primary. Choose the active one, or edit your own.

    Connect an agent

    Point any coding agent at Caucus. The agent believes it's talking to one model; Caucus convenes the panel behind the scenes.
